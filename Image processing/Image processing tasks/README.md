# Image Processing Tasks

Hello! 👋 Welcome to the **Image Processing Tasks** project! This repository contains Python code for performing various image processing techniques, especially edge detection. The code helps in identifying significant changes in image brightness, which is useful in tasks like pattern recognition, feature extraction, and more.

## What does this project do? 🤔

This project allows you to:
- Detect edges in images using filters like Prewitt, Sobel, and Canny algorithms.
- Apply Gaussian blur to remove noise before processing.
- Compare different edge detection methods and their results.
