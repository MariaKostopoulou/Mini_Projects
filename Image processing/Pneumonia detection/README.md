# Pneumonia Detection Project

Hello! 👋 Welcome to the **Pneumonia Detection** project! This repository contains code to help detect pneumonia from chest X-ray images using machine learning models. It leverages deep learning techniques and various image processing steps to classify X-ray images as "Normal" or "Pneumonia."

## What does this project do? 🤔

This project allows you to:
- Train and test a deep learning model for pneumonia detection using chest X-ray images.
- Preprocess and augment the image data to improve model accuracy.
- Generate confusion matrices and classification reports for performance evaluation.
- Visualize the training process (accuracy and loss over epochs).
