# Image Enhancement Project

Hello! 👋 Welcome to the **Image Enhancement** project! This repository contains some awesome Python code to help you enhance images and analyze them, making them clearer or adjusting their quality. The project also includes generating histograms to see how the brightness and contrast of an image are distributed.

## What does this project do? 🤔

This project helps you:
- Enhance images by adjusting their brightness and contrast
- Generate histograms to visualize the distribution of tones in the image
- Compare original images with their enhanced versions
